插件安装/使用方法
将本插件上传至Emlog中的插件文件夹子中（或者直接去应用后台下载） 然后去后台开启插件，对插件进行相应的配置。 大功告成！您可以正式开始使用插件了！ 您可以前往撰写文章页面，体验强大的ai给您带来的写作灵感！ 仅需在你的博客后台即可使用！
